# Demo corpus — Israeli labor & employment rights

⚠️ **These are simplified, illustrative demo documents** written for the Asmakta
demo. They summarize well-known, publicly-established Israeli labor-rights topics
in plain language. **They are not legal advice and are not authoritative texts.**

The eval suite measures whether the assistant answers **faithfully from this
corpus** and **refuses** when the answer isn't here — so the fixtures' gold
answers are consistent with the statements in these files by construction. Before
any real-world use, replace these with authoritative sourced texts (e.g. official
statutes / Kol-Zchut articles) and re-derive the fixtures.

Files: English (`*.en.md`) and Hebrew (`*.he.md`) covering distinct topics so the
per-language eval breakdown is meaningful.
